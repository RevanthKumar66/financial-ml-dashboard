# 📊 Financial ML Dashboard

A premium, high-fidelity financial intelligence platform built with **Streamlit** and **Machine Learning**. This dashboard provides real-time market analysis, automated engineering pipelines, and predictive modeling for financial assets.

## 🚀 Key Features

- **Intellectual Analytics Hub**: Beautifully designed dashboard with real-time performance tracking and statistical insights.
- **Automated ML Pipeline**: A robust engineering architecture that standardizes data, handles missing values, and prepares features for modeling.
- **Advanced Forecasting**: 
  - **ARIMA**: High-precision statistical forecasting for time-series data.
  - **Random Forest & Linear Regression**: Predictive models with automated validation metrics (RMSE, R²).
- **Model Management**: Export trained intelligence models directly as `.pkl` files and get ready-to-use deployment code.
- **Premium UI/UX**: Minimalist, professional interface featuring Material Icons, glassmorphism-inspired cards, and high-performance Plotly visualizations.

## 🛠 Tech Stack

- **Core**: Python 3.12+
- **Frontend**: Streamlit
- **Visualization**: Plotly, Seaborn, Matplotlib
- **Machine Learning**: Scikit-Learn, Statsmodels
- **Data Handling**: Pandas, NumPy

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/RevanthKumar66/financial-ml-dashboard.git
   cd financial-ml-dashboard
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   streamlit run app.py
   ```
